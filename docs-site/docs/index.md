## What is ChefByte?

ChefByte is an all-in-one kitchen management system that unifies:

- **Inventory tracking** with barcode scanning
- **Meal planning** with macro optimization
- **Recipe management** with nutrient calculations
- **Shopping lists** with Walmart price integration
- **Macro tracking** with real-time progress

## Quick Start

1. **Try the Demo**: Click the button above to explore a pre-populated account
2. **Create Account**: Sign up with your email to start fresh
3. **Add Products**: Scan barcodes or manually create items
4. **Plan Meals**: Build weekly meal plans optimized for your macro goals
5. **Track Progress**: Watch your nutrition dashboard update in real-time
