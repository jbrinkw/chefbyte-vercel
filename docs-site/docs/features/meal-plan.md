## Overview

The meal planning system enables:

- **Weekly Calendar**: Plan meals for each day
- **Recipe Integration**: Add recipes with full macro info
- **Meal Prep Mode**: Batch cook for future meals
- **Macro Preview**: See daily totals before eating
- **Execution Tracking**: Mark meals complete for macro logging

## Adding Meals

### From Recipes

1. Go to Recipes page
2. Find desired recipe
3. Click "Add to Meal Plan"
4. Select date and meal type
5. Recipe appears in calendar

### Quick Add

1. On Meal Plan page, click + on a day
2. Select recipe or product
3. Set portion size
4. Choose meal type (Breakfast, Lunch, Dinner, Snack)

### Product-Based Meals

For simple meals without recipes:

1. Click + on desired day
2. Select product directly
3. Set amount consumed
4. Product macros count toward daily total

### How It Works

1. Add recipe to meal plan
2. Enable "Meal Prep" toggle
3. Entry shows as meal prep (not regular meal)
4. Click "Execute" to batch cook
5. System consumes ingredients and creates [MEAL] products
6. [MEAL] products appear in inventory for future consumption

### Meal Prep vs Regular

| Aspect | Regular Meal | Meal Prep |
|--------|--------------|-----------|
| Mark Done | Consumes immediately | Creates [MEAL] products |
| Macros Count | Same day | When [MEAL] consumed |
| Inventory | Ingredients deducted | Ingredients deducted, meals added |
| Use Case | Eating now | Cooking for later |

## Marking Meals Complete

When you eat a planned meal:

1. Find the meal entry
2. Click the checkbox or "Mark Done"
3. Macros add to your daily total
4. Entry shows as completed

!!! tip "Real-Time Tracking"
    Mark meals done immediately after eating for accurate daily macro totals on the Home dashboard.

## Daily View

Each day shows:

- All planned meals grouped by type
- Running macro total for the day
- Meal prep entries (highlighted differently)
- Completion status for each item

## Week Navigation

- Use arrows to move between weeks
- Today is highlighted
- Past days show historical data
- Future days show planned meals

## Macro Preview

Before eating, see projected macros:

- View planned items for any day
- See total calories, protein, carbs, fats
- Compare to your daily goals
- Adjust portions if needed

## Best Practices

1. **Plan Weekly**: Schedule meals every Sunday
2. **Batch Prep**: Use meal prep mode for efficient cooking
3. **Mark Promptly**: Update meals as you eat them
4. **Balance Macros**: Spread protein across meals
5. **Leave Flexibility**: Don't over-plan every meal

## Shopping Integration

The meal plan drives smart shopping:
